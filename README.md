## How It Works

1. **Export from Snapchat:**
   - Open Snapchat ‚Üí Settings ‚Üí Privacy ‚Üí Export your data
   - Select "Memories" and request your data
   - Wait for Snapchat to prepare your backup (usually 24-48 hours)
   - Download the ZIP file(s)

2. **Import to SC Memories Saver:**
   - Open this app and tap "Import ZIP Files"
   - Select the ZIP file(s) from your downloads
   - The app will extract and organize your memories

3. **Browse & Manage:**
   - View all your memories in the gallery
   - Tap to see details of each memory
   - Export selected memories to your Photos library

## Troubleshooting

### "Could not read the ZIP file" error
- Ensure you downloaded the ZIP directly from Snapchat
- Try importing one ZIP file at a time
- Check that the file isn't corrupted

### App crashes when importing large files
- Try importing one ZIP file at a time instead of multiple
- Restart the app and try again
- Free up device storage space

### "Product not available" when trying to unlock premium
- Make sure you have an active internet connection
- Restart the app
- Check that you're signed into the App Store
